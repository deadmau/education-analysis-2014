Tweet Sentiment Analysis

credit to: http://ravikiranj.net/drupal/201205/code/machine-learning/how-build-twitter-sentiment-analyzer

Classifies tweets by either “positive”, “neutral”, “negative” using Naive Bayes Classifier provided by nltk.

Provided small sample, specifically 490 tweets, manually classified by 3 category above.

Usage:
	Train samples by running “sample_trainer.py”. This will create a Bayes Net model 
	to classify new tweets.
	To analyze new tweet, run “sentiment_classifier.py”
